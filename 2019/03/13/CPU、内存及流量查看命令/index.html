CPU、内存及流量查看命令

获取应用pid

adb [-s device_serial_number] shell ps | grep package

输出的第二列即为该包名对应的pid

获取CPU消耗

获取CPU总消耗

adb [-s device_serial_number] shell cat /proc/stat

获取特定进程CPU消耗

adb [-s device_serial_number] shell cat /proc/pid/stat

第14-17列分别为（参考文档）：

  • utime 该进程处于用户态的时间 单位 jiffies。
  • stime 该进程处于内核态的时间
  • cutime 该进程等待处于用户态子进程的时间
  • cstime 该进程等待处于内核态子进程的时间

获取内存占用

adb [-s device_serial_number*] shell dumpsys meminfo *packagename

取 含TOTAL关键词那一列作为进程的总内存消耗。

获取流量消耗

adb [-s device_serial_number] shell cat /proc/pid/net/dev
