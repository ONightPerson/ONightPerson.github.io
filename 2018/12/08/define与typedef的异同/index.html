#define与typedef的异同

Posted on 2018-12-08 | In C

#define

#define 是 C 指令，用于为各种数据类型定义别名。

如：

typedef

顾名思义，typedef用来为类型取别名，它是 C 语言提供的关键字。

如:

#define VS typedef

相同点

两者都可以为各种数据类型定义别名

不同点

1. typedef 仅限于为类型电仪符号名称，#typedef不仅可以定义别名，还能为数值定义别名，如`#define TRUE  1`;

2. typedef 是由编译器执行解释的，#define语句是有预编译器进行处理的。
