csv安装

sudo pip install csvkit
若 pip 安装，可使用下面命令安装
sudo easy_install pip（macOS)
sudo apt-get install phthon-pip (Debian or Ubuntu)
sudo yum install python-pip (Fedora)

使用

  • Excel 转 CSV

in2csv data.xls > data.csv

  • JSON 转 CSV

in2csv data.json > data.csv

  • 打印行号
    csvcut -n data.csv
